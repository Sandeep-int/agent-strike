# Security Policy

## About Agent Strike

Agent Strike is an internal adversarial testing tool for Agent Shield.
It fires attack prompts at the Shield API to measure detection accuracy.
This repo contains no production endpoints, no user data, no public API.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ Yes    |

## Reporting a Vulnerability

If you find a security issue in this repo, report it privately:

**Email:** sandeep@agent-shield.dev  
**Do not** open a public GitHub issue for security vulnerabilities.

Expected response time: 48 hours.
